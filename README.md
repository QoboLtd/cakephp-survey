About
-----

Template for building CakePHP 3 plugins.

Developed by [Qobo](https://www.qobo.biz), used in [Qobrix](https://qobrix.com).

Usage
-----

Pull the template code into your plugin:

```
composer require qobo/cakephp-survey
```
